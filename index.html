FAQ

Frequently Asked Questions

Is my data safe with Drive Purge?

Absolutely. Drive Purge runs entirely on your local machine. We use OAuth 2.0 for authentication and all tokens are encrypted at rest using Fernet encryption. No file content is ever sent to external servers — only Google's official Drive API is contacted.

Can I undo a deletion?

Yes. Files are moved to your Google Drive Trash by default, so you can restore them at any time. Drive Purge also includes a built-in Restore panel for quick undo operations.

Does it work with Google Workspace accounts?

Yes, Drive Purge fully supports Google Workspace (business) accounts, provided your admin has authorized the Google Drive API for your organization.

What platforms are supported?

Drive Purge runs on Linux (AppImage), macOS (.app), and Windows (.exe). It requires an internet connection for Google Drive API access.

Why does the scan take a while?

Google Drive API has rate limits (1,000 queries per 100 seconds). Drive Purge respects these limits with configurable request delays to prevent quota exhaustion while maximizing throughput.
